Simple Example

This is a simple example of the usage of NumCpp.

First a vector of 16 elements is initialized with ones. Then a one-dimensional FFT of x is calculated. Then the vector is reshaped into a 4x4 matrix and printed to the console.

int main()
{
Vector<double> x = ones(16);
auto y = fft(x);
auto z = reshape(y,4,4);
print(z);
}
