Detailed Description

RGBA color type.

It consists of four doubles decribing the color in RGBA colorspace. For standard colors, a string constructor exists that allows to specify colors as strings. Here are some example color definitions:

color_t c = "blue";
color_t f = {1.0, 0.5, 1.0. 1.0};
color_t h = color.green;
h.r = h.g = h.b = h.a = 1.0;
